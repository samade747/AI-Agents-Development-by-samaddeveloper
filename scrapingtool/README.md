# ScrapingTool - Web Scraping AI Agent

This Streamlit app uses OpenAI models and ScrapeGraph AI to scrape data from websites. You can specify the URL and a prompt for the AI to extract relevant information.

## Requirements

- Python >= 3.8
- Streamlit
- ScrapeGraph AI
